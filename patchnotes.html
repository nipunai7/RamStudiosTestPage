  • Update Note 2019 August 19 v1.04

    • Patch Type - Steam Patch
    Date :- 2019/08/19

    Update Note v1.04

    We have fixed so many bugs and also improve a lot of things. You guys have been so much help for us and we will continue to update our game. Make sure you give us your feedback. Read the full patch note below. You can also find a lite version of this patch note in-game.

    - New Features -

    • New Europe server added.
    • In-game text chat added (Press "Enter" in-game) -  
      • Now you can chat with others In-game this is one of the most requested features you guys been asking from us.
    • New graphics option added (Visual Effects) -  
      • Now you can turn off Visual effects to improve performance.
    • Low quality lighting mode added -  
      • Now you can turn off lighting in maps to improve performance press "Backspace" in-game.

    - Sound Effects -

    We heard from you! yep, it's a Pune. Sounds play a huge role in a shooting game special on a big map so we improved sounds and hit effects to give you more feedback.
    • Improved weapon sounds -
      • We added new sounds to each weapon.
    • Improved gun turrets shooting sounds -
      • We added a new sound effect for gun turrets and improve sound quality.

    - Gameplay Improvements -

    • Bullet speed increased in every weapon.
    • Decrease FPP mode weapons reloading time to match up with TPP mode.
    • lobby time decreased. (from 5 minutes to 1 minute)
    • Nitro Boost price increased. (from 200 coins to 600 coins)
      • Some players like Nitro Boost and its speed but some don't like it we increased its price so it can't easily obtain. We will keep monitoring how this powerup impacts gameplay.

    - UI & UX improvements and Bug Fixes -

    • Ultrawide screen issue fixed.
    • Minor UX Improvements.

    - Map improvements and Bug Fixes -

    • The Town map and Islands Map performance improved.
      • There were lag spicks in these maps. Give us your feedback if you experience any issue in these maps.
    • Respawn points changes.
      • Players now spawn from the XRONIN Ships (the spaceship).

    - Bug Fixes -

    • Fixed an issue where Crosshair was visible while ADS in FPP mode.
    • Fixed an issues where lobby time is not decreasing when players left the lobby.
    • Fixed an issue where players can't run while ADS.
    • Players spawn points and Coin Drops points adjusted in lager maps.

    - Known Issues -

    • Mouse sensitive cannot be changed in the main menu.
    • Mouse sensitive automatically changes.
    • Fixed an issue where players can't run while ADS.
    • Unexpected FPS drops. You can fix this visit here to see how. This is a tempory solution we are looking into it.

    - Removed -

    • Knockdown Feature removed.

    Keep Us Updated

    If you found any other Issues report to us. Thank you!
